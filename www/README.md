# samhwasnd 

Server: PHP Version 7.3.x

## Install

```sh
git clone https://github.com/pelogvc/signstand.git
chmod -R 755 signstand
cd signstand
cd data
mkdir session
cd ../
chmod -R 707 data
```

## Development

- hosts 파일에 해당 내용 추가
```sh
127.0.0.1 signstand.doto.li
127.0.0.1 samhwasnd.doto.li
```
- 본점: samhwasnd.doto.li
- 지점: signstand.doto.li
