# mall-eroumcare 

Server: PHP Version 7.3.x

## Install

```sh
git clone https://github.com/pelogvc/mall-eroumcare.git malleroumcare
chmod -R 755 malleroumcare
cd malleroumcare/data
mkdir session
mkdir cache
cd ../
chmod -R 707 data
```

## Development

- hosts 파일에 해당 내용 추가
```sh
127.0.0.1 mall.eroumcare.doto.li
```
